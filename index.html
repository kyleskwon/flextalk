Setting the main axis with flexDirection
The main axis defaults to ‘column’ (from top to bottom), but the other option is ‘row’ (left to right)
Spacing along the main axis with justifyContent
Once the main axis is determined, justifyContent can be used to space the children along that axis. Options include flex-start, flex-end, center, space-around, and space-between. Note: rather than justify-content (expected in CSS), React-Native uses camelCase in the stylesheet
Spacing along the cross axis with alignItems
The cross axis is the complementary axis to the main axis. If the main axis is set in the ‘column’ direction, then the cross axis is the ‘row’ direction. If the main axis is ‘row’, then the cross axis is ‘column’. Similar to justifyContent, we can use alignItems to set the spacing as flex-start, flex-end, or center
Intermediate Tips
Positioning a black sheep with alignSelf
If we want a single child element to be positioned differently along the cross axis, then alignSelf comes in handy. Let’s say we want all of the child elements to centered along the horizontal cross axis, but the first child element to be left aligned. We can assign it alignSelf: flex-start to achieve this effect
